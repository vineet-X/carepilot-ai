# CarePilot AI

## AI-Powered Personal Healthcare Assistant

CarePilot AI is an intelligent healthcare assistant that helps individuals understand their medical reports using AI.

### Features
- 📄 Upload medical reports (PDF)
- 🤖 AI-powered report analysis
- 💬 Chat with your medical reports
- 📈 Track health history
- ⚠️ Highlight abnormal values
- 🍎 Personalized health recommendations
- 🔒 Secure authentication with Supabase

### Tech Stack
- Next.js
- TypeScript
- Tailwind CSS
- Supabase
- Google Gemini API

### Target Users
Patients who want to better understand their medical reports without medical jargon.
